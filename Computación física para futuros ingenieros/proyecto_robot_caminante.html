Proyecto robot caminante

Problema técnico

Introducción:

Vamos a tratar de construir un robot con patas e inteligencia capaz de evitar objetos a su paso. Los robots bípedos son realmente muy difíciles de llevar a cabo puesto que su dinámica es muy compleja. Por eso buscaremos soluciones más simples que se pueden hacer con cuatro patas. Para extraer ideas podemos visitar las siguientes paginas de Instructables:

http://www.instructables.com/id/How-to-make-a-one-motor-walker-without-bicore/ Todo un prodigio minimalista y mecánico. No requiere de placa controladora porque es puramente mecánico y no tiene capacidad para evitar obstáculos pero de este ejemplo se pueden sacar muy buenas ideas.

http://www.instructables.com/id/Mobile-Robotics-with-Scratch-Build-an-Arduino-base/?lang=es. Primera versión de lo que se quiere conseguir. Muy fácil de montar si se dispone de piezas tipo meccano. En realidad

http://www.instructables.com/id/Simple-Walker-Robot/?lang=es Divertida opción de robot cuadrúpedo que evita objetos.

Planteamiento del problema:

Diseñar, construir un robot caminante de cuatro patas e implementar el programa para que evite los obstáculos.

Retroalimentación

  • No se tienen porque usar los mismos materiales que los que aparecen en cada caso de Instrutables.
  • Se puede reusar material como plásticos de botes de colonia, detergente,... o metal de latas, cajas de disqueteras viejas,... (¡Ojo! respetar las normas de seguridad para evitar cortes) cartón de tetrabricks o de cajas, palillos de oídos o de chupachus para hacer ejes o tubos,... Todo aquello que podamos tener en casa para poder construir el chasis del robot.
