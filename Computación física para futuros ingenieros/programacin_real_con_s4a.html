Programación real con s4a

El primer programa que se muestra de ejemplo para Arduino se denomina BLINK y consiste en montar el circuito del LED intermitente en una placa real de Arduino. En el ordenador se debe cargar el programa en el Arduino IDE y descargarlo en la placa conectando el ordenador y Arduino mediante un cable USB. Nosotros evitaremos la programación en Wiring porque usaremos s4a pero es conveniente comparar la sintaxis  de los dos mundos:

Wiring en Arduino IDE s4a
/*
  Blink
  Turns on an LED on for one second, then off for one second, repeatedly.
 
  This example code is in the public domain.
 */

void setup() {                
  // initialize the digital pin as an output.
  // Pin 13 has an LED connected on most Arduino boards:
  pinMode(13, OUTPUT);     
}

void loop() {
  digitalWrite(13, HIGH);   // set the LED on
  delay(1000);              // wait for a second
  digitalWrite(13, LOW);    // set the LED off
  delay(1000);              // wait for a second
}
