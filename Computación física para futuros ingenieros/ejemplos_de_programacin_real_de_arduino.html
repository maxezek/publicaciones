Ejemplos de programación real de Arduino

Para programar y comunicarse con Arduino se usa el entorno de desarrollo Arduino IDE. Es un entorno con una ventana gráfica pero en texto.

Nosotros no usaremos este entorno de programación por resultar poco intuitivo pero es necesario tenerlo para usar otras posibilidades gráficas más intuitivas como Scratch for arduino, s4a.

Ambos, cuando se instalan en MAX se encuentran en el menú programación.

En Arduino IDE disponemos de mucho código de ejemplo. Podéis instalaros Arduino IDE en casa aunque no tengáis la placa arduino. Ya sabéis que nosotros recomendamos MAX y con MAX se puede instalar desde synaptic (ver bloque 13 del curso, MAX: distintas instalaciones).

Si ejecutáis Arduino IDE podéis ver el ejemplo de la programación real del LED intermitente yendo a:

File > Examples > Basics > Blink

El código lo veréis en una nueva ventana y se puede compilar (comprobar y generar un archivo binario ejecutable en la placa arduino) haciendo clic en el botón Verify ,  a. Si conectamos el PC con la placa por USB entonces pulsaríamos en Upload, →.

El código de blink es este:

/*
  Blink
  Turns on an LED on for one second, then off for one second, repeatedly.
 
  This example code is in the public domain.
 */

void setup() {                
  // initialize the digital pin as an output.
  // Pin 13 has an LED connected on most Arduino boards:
  pinMode(13, OUTPUT);     
}

void loop() {
  digitalWrite(13, HIGH);   // set the LED on
  delay(1000);              // wait for a second
  digitalWrite(13, LOW);    // set the LED off
  delay(1000);              // wait for a second
}
