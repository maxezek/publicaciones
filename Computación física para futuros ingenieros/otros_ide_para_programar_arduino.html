Otros IDE para programar Arduino

Un IDE o entorno de desarrollo integrado es un aplicación que aporta un entorno para facilitar el desarrollo de programas de ordenador. Contiene un editor de código fuente, un depurador y en muchos casos un interprete y/o un compilador según corresponda. Los más sencillos aportan un entorno de consola o texto y los más atractivos visualmente incluyen un entorno gráfico agradable.

En MAX se dispone de IDEs como: gedit, geany, SciTe, Bluefish,... y se pueden instalar los potentes eclipse o netbeans. Cabe la posibilidad incluso de disponer de un RAD o software de desarrollo rápido de aplicaciones en los cuales se incluye en el propio entorno la posibilidad de hacer funcionar el programa que se está diseñando. Entre esos casos tenemos Gambas 3, un RAD y lenguaje de programación muy similar a Visual Basic .NET.

Scratch y Scratch4Arduino, s4a, podemos considerarlos RAD. Son lenguajes de programación visual por bloques cuya programación es orientada a eventos y dentro de su propio entorno podemos ejecutar el programa. Su mayor ventaja es que el sistema de programación por bloques visuales es muy intuitivo, sencillo de manejar y construye los programas obligatoriamente de manera sintácticamente correcta. Por estas razones su uso en la enseñanza es muy extendido.

Pero tiene sus limitaciones. Un proyecto como el propuesto es difícil de implementar con Scratch y sus variantes. Afortunadamente hay otros lenguajes e IDEs. Unos son visuales y otros utilizan código fuente en texto. A continuación vamos a repasar varios de ellos con los que se puede programar Arduino y otras placas.
