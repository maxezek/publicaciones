Simulación de Arduino

Arduino es una placa controladora de hardware libre. El hardware consiste en una placa con un microcontrolador sencillo y bajo coste y puertos de entrada/salida.  Los puertos son pines o patillas para hacer conexiones eléctricas con cables. Tiene varios pines digitales y 6 entradas analógicas. Con ello, se pueden conectar sensores o actuadores y se pueden controlar luces, motores y otros actuadores.

Que sea hardware libre significa que podemos montar y reconfigurar el diseño de la placa controladora sin depender de terceros. Por ello existen varias plataformas Arduinos y de ellas nos vamos a centrar en la más común, el arduino UNO.

Por otro lado el software consiste en un entorno de desarrollo denominado Arduino IDE que implementa el lenguaje de programación Processing/Wiring y el cargador de arranque que es ejecutado en la placa. Como alternativa a este IDE y al lenguaje processing se puede usar el software s4a (scratch for arduino) o s2a-fm (scratch to arduino - firmata).

Antes de usar estas variantes de Scratch vamos a simular la placa arduino con una imagen y hacer implementaciones virtuales de lo que se puede programar. Vamos a simular la programación de un LED para que sea intermitente, el control de giro de un motor CC simple y lo mismo para un servo motor.

Todas las imágenes de Arduino que son dibujos están exportadas del programa fritzing.
